## 8. 완료 조건

AI는 구현 후 아래 항목을 자체 검증해야 함.

* `npm run build` 성공
* TypeScript 타입 에러 없음
* dead code 없음
* 새 코드가 실제 화면에서 사용됨
* 기존 기능이 깨지지 않음
* 작업 제외 범위를 침범하지 않음
* 검증하지 못한 항목은 “성공”이 아니라 “미검증”으로 표시

## 9. 문서 생성 금지 조건

* `.ai-docs/issues/` 문서 생성
