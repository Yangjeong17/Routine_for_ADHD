# 추가 구현
## 겹치는 일정카드에 대해
일정의 시간이 너무 짧으면 바로 이어지는 일정과 카드가 겹쳐보이는 이슈가 발생.
### 이슈 해결 방안
지금처럼 겹쳐보이게 하되, 현재시간 수평선 구현되면, 현재 시간의 일정카드가 가장 앞으로 오도록 하게 만들어줘.
