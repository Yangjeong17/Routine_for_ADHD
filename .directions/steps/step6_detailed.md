## 카테고리 한국어 변환
카테고리는 json에서 영문이든 한글이든 상관없이 만들되, 앱에서 보일때는 한글만 보이게 하고 싶음. 지금은 아직도 영문으로만 나옴. -> 추후 앱 전체를 영문/한글 선택할 수 있게 만들거고, 그때는 사용자가 language를 English로 선택할경우, json파일에서 만든게 영문이든 한글이든 앱에서 보일때는 영문으로 보이도록 만들고자 함.

# 추가 구현
## 일정 추가 버튼
현재 하단의 모든 요일에 +버튼이 있는데 오른쪽 하단에 호버링 되는 버튼 활성화 해줘.
이 버튼은 스크롤을 올리거나 내려도 항상 그 자리에 위치하도록 구현 필요함.

## 색상 변경
### 현재 이슈
현재 일정카드의 수정버튼을 누르면 "루틴 블록 수정" 창이 뜨는데, 여기에서 "중요도" 색상 바꿀수 있는거처럼 보이지만 선택해도 메인 화면에는 반영되지 않음.
### 추가 구현1
"중요도" 뿐만 아니라 "카테고리"의 색상도 변경할 수 있도록 만들어줘.
### 추가 구현2
사용자가 새로 사용한 색상은 계속 사용하고 싶을꺼야. 그렇기 때문에 이 색을 저장하고 나중에도 사용할 수 있도록 만들어야 될거 같음.
사용자가 "카테고리"의 색을 변경했다면 플래너의 모든 색을 그 색으로 변경하고, 앞으로 해당 카테고리의 색은 그 색이 디폴트가 되도록 만들어줘.

## 카테고리 수정
일정카드의 수정버튼을 눌렀을때 보이는 간단 편집창의 경우에는 드롭박스로 잘 구현되어 있지만
간단 편집창에서 다시한번 수정버튼을 누르면 나타나는 "루틴 블록 수정" 창에서는 카테고리를 수정하려면 텍스트를 입력해야됨. -> 카테고리를 수정할 수 있도록 한건 좋은데, 이렇게 되면 기존에 동일한 카테고리로 설정되어 있던 일정카드들을 모두 선택해서 수정해줘야 함.
이 부분 드롭박스 방식으로 바꿔서 사용자가 선택할 수 있도록 하고, 드롭박스에서 수정이나 추가할수 있도록 구현하는게 더 좋을거 같음. 
그리고 앱에서 보이는 방식이 한글이면 수정할때 드롭박스도 한글로 만들어줘.

### 추가 수정 1

1. 드롭박스화 되긴 했지만, 다른 카테고리가 보이지 않음.
카테고리가 너무 많아서 안보이는거면, 기본 카테고리를 설정하고 싶음.
- 기본 카테고리: 휴식, 업무, 식사, 운동, 약속(이건 예를들어 친구들과의 약속이 일정에 있다면 선택하는 것), 루틴(스트레칭, 명상 등 주기적으로 진행하는 다양한 루틴을 이 카테고리에 넣을거임) 이정도로 구현했으면 좋겠음.

2. 아직도 수정창에 카테고리가 영어로 되어 있는 이슈
수정창에 카테고리가 영어로 되어 있고 드롭다운 하면 한글로 동일한 카테고리가 보임
ex. rest -> (드롭다운) -> 휴식
이걸 위에 1번 항목처럼 기본 카테고리가 "한글로" 보이게 만들고, 추가 또는 각 카테고리 명을 수정할 수 있도록 구현하기.

### 추가수정 2

1. 루틴블록 수정 창에서 카테고리의 내용을 지워야 드롭박스의 다른 카테고리를 볼 수 있음. 처음부터 드롭박스 보이게 만들고 디폴트를 버튼선택으로 만들기. 그런 다음 "수정"이나 "추가" 버튼을 누르면 그때 텍스트를 작성할 수 있도록 만들기.

2. 카테고리를 수정한 경우, 컬러 반영이 되지 않음. 
예를 들어 운동 -> 식사로 변경한 경우 카테고리명은 변경된게 확인되지만 초록->노랑으로 컬러변경은 없음.

3. 카테고리의 컬러 선택이 불가능함
1번 항목에서 카테고리의 수정/추가 할 수 있게 구현할때 컬러 설정도 가능하도록 만들어줘.
