# steps/cowork/

Cowork(대화형 Claude)와 나눈 대화를 정리해 **Claude Code(코드 에이전트)에게 작업을 지시할 때** 사용하는 문서를 보관한다.

## 용도

- Cowork와 논의·분석한 내용을 코드 작업 지시문 형태로 정리한다.
- Claude Code에게 그대로 전달할 수 있는 작업 브리프를 만든다.
- 즉, `steps/`의 자연어 기능 지시와 실제 코드 구현 사이를 잇는 "지시 정리본"이다.

## 파일명 규칙

- `step[N]_cowork_기능명.md` (예: `step7_cowork_drag_edit.md`)
- 영문 소문자 + 언더스코어 사용, 한글 파일명 지양

## 워크플로우에서의 위치

1. Cowork와 대화하며 작업 방향·범위·주의사항을 정리한다.
2. 그 결과를 이 폴더에 step별로 저장한다.
3. Claude Code에게 해당 문서를 전달해 구현을 지시한다.
